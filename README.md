# Prompt-simula-o-entrevista-vaga-dev-java-junior
Este repositório contém a análise estruturada de uma vaga para **Desenvolvedor Java Júnior**, realizada como parte de um trabalho de curso.   O objetivo é apresentar de forma clara os principais pontos da vaga — título e propósito, senioridade, stack tecnológica e soft skills — organizados em formato de documentação para consulta e aprendizado.
